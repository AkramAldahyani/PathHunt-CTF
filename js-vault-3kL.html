Base64 JavaScript

Category: Decoding · Technique: Read inline JS + decode Base64
Brief

Developers love to "hide" strings by encoding them in Base64. It's trivially reversible — anyone with a browser console can decode it in one line. Never rely on Base64 for security. It's obfuscation, not encryption.

Two variables are waiting for you at the bottom of this page's inline <script>. Decode them and you've got the flag and the next path.

Hint In the browser console, atob('...') decodes a Base64 string. btoa('...') encodes one. Paste each variable and run it.
Console walkthrough

Open DevTools (F12) → Console tab. Type:

atob('ZmxhZ3tkZWNvZGVfbWV9') atob('L3JvYm90LXpvbmUuaHRtbA==')

Each atob call returns the decoded string. One is your flag, the other is the path to Challenge 04.

CSS Leak

Category: Recon · Technique: Stylesheet inspection
Brief

// Not everything visible is in the HTML.

Web pages load many files besides the HTML document — stylesheets, scripts, images, fonts. Each of those files is fetched by your browser and sits in the Network tab of DevTools. Developers sometimes leave secrets in CSS comments, mistakenly thinking "nobody looks at stylesheets."

This page loaded two stylesheets. One is the main theme. The other holds the clue.

Hint Open DevTools → Network tab → reload the page. Watch for .css files. Click each and read the response. Or open <head> and check every <link rel="stylesheet"> tag.
Inspection technique

CSS files are plain text. Comments in CSS use /* ... */ syntax and are stripped out during rendering — your browser reads them, but they never appear on screen.

pro-tip  Always scan every resource the page pulls in. The flag you need may not live where the page is rendered.

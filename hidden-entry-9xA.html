CSS Leak

Category: Recon · Technique: Stylesheet inspection
Brief

// Not everything visible is in the HTML.

A page is rarely a single file. The HTML you read is just the entry point — the browser then fetches everything that document references. Developers sometimes hide notes in those side files, betting nobody bothers to look past the markup.

Hint The HTML document is not the only thing the server sent you. Whatever else loaded with it is plain text too. Count the resources. One does more than it claims.
Inspection technique

Every file the browser pulls in is fair game. Markup, scripts, fonts, styles — they all travel as plain text. Anything that looks like a comment in those files is invisible on screen, but right there in the response body.

pro-tip  If a flag isn't in the page you can see, it's in something the page made you fetch.

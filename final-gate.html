Brief

You've made it to the last gate, hunter. One more decode stands between you and the victory flag. Everything you've learned — reading source, using the console, decoding Base64 — comes together here.

There are two encoded payloads in the script at the bottom of this page:

  • final — a single Base64 string. The easy path.
  • vault — Base64 over XOR with a key. The hard path, same flag.

Either one lands you the flag. Pick the one you feel like solving.

Easy path In the console: atob(final)
Hard path (optional) Base64-decode vault, then XOR each byte with the repeating key "pathhunt". The snippet is provided in the console hints below.
Console snippets

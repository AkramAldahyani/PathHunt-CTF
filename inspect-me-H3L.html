Challenge 01 · Easy · 100 pts

HTML Source

Category: Recon · Technique: View Source
Brief

Welcome, hunter. Every web page hands you its own source code — you just have to ask for it. Developers often leave comments, TODOs, or debugging notes that were meant to be removed before launch. Sometimes they aren't.

This page is your training wheel. The flag is somewhere in the HTML source of the very page you're looking at. The trail to the next page is there too.

Hint Right-click anywhere on the page → View Page Source (or press Ctrl+U / ⌘+Option+U). HTML comments look like <!-- ... --> and are invisible in the rendered page.
What you're looking for

Two things on every challenge page:

  1. A flag in the format flag{something} — submit it on the Submit page.
  2. A next path that tells you which URL to visit for the next challenge.

When you find the flag, head to /submit.html to capture it.
